# vue-ssr-sample

Vue.jsでSSRしてみるサンプル.

[vue-hacker-news-2.0](https://github.com/vuejs/vue-hackernews-2.0)
を簡単化したもの.

vuejs
vue-router
vuexを主に使い.

* SSRをして表示する方法.
*  サーバ側でAPIコールして動的なページを最終的にレンダリングする方法

などを示す.

解説記事　[Qiita](http://qiita.com/namazu510/items/e699afb2fb8161cbac2e)

# 実行
```bash
# install
yarn  # or npm install
npm run duild
npm start # lunch server on 8080
```
